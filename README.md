## Dallas Airbnb Market Analysis

Analyzed 48,711 Dallas Airbnb listings across 17 neighborhoods 
using Python to identify pricing patterns and revenue drivers.

### Key Findings
- Occupancy rate is strongest revenue predictor at 0.22
- Lake Highlands top revenue neighborhood at 4,141 USD
- Peninsula highest occupancy at 32.8 percent
- Proposed neighborhood based dynamic pricing tool

### Tools
Python, Pandas, Matplotlib, Seaborn, Google Colab
